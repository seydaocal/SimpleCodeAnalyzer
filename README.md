Simple Code Analyzer
This project is a simple code analyzer developed in C#.
It scans each cs files in a given folder and analyze the structure. 
It counts 
- total lines of code
- comment lines
and detects usage of keywords:
  - if
  - for
  - while   (More keywords can be added.)

How it works:
1. The user enters a folder path.
2. Program checks if the folder exists.
3. Each cs files in the folder is read line by line.
4. Program analyzes each line and prints statistics.

Technologies used:
- C#
- .NET
- File I/O
- Dictionary data structure

This project was designed to:
- understand how the real tools scan source code
- gain experiences with collections and files processing
- learn how software analyzes projects work.

This is a student project developed for learning purposes.
