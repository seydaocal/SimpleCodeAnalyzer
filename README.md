Simple Code Analyzer
This project is a simple code analyzer developed in C#.
It scans each cs files in a given folder and analyze their structure. 
It counts 
- total lines of code
- comment lines
It also detects usage of keywords:
  - if
  - for
  - while
  (More keywords can be added.)

How it works:
1. The user enters a folder path.
2. The program checks if the folder exists.
3. All cs files in the folder are read line by line.
4. The program analyzes each line and prints statistics.

Technologies used:
- C#
- .NET
- File I/O
- Dictionary data structure

This project was desinged to:
- understand how the real tools scan source code
- gain experiences with collections and files processing
- learn how software analyzes projects work.

This is a student project developed for learning purposes.
